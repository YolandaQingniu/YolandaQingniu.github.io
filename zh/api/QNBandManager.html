QNBandManager

QNBandManager

cancelBind

取消绑定设备

参数

名称 类型 说明
callback QNResultCallback 安卓是接口，IOS用闭包。返回解绑的结果

checkSameBindPhone

确认是否绑定该手机

该方法主要用于检测手环上次绑定的手机是否与当前绑定的手机相同。如果不相同，证明手环上次绑定的手机不是当前手机，因此在判断不相同时可以对手环进行重新设置

参数

名称 类型 说明 泛型类型
callback QNObjCallback 返回检测的结果 android：Boolean
iOS：NSNumber

fetchBandInfo

获取手环的固件版本、软件版本、电量

参数

名称 类型 说明 泛型类型
callback QNObjCallback 返回获取到的数据 QNBandInfo

syncBandTime

设置手环的时间

参数

名称 类型 说明
date Date 当前时间
callback QNResultCallback 返回设置的结果

syncAlarm

设置闹钟详情

目前最多支持10个闹钟

参数

名称 类型 说明
alarm QNAlarm 闹钟
callback QNResultCallback 返回设置的结果

syncGoal

设置运动目标

参数

名称 类型 说明
stepGoal int 运动目标
callback QNResultCallback 返回设置的结果

syncUser

设置用户信息

参数

名称 类型 说明
user QNUser 用户信息，主要包括用户id、体重、身高、生日
callback QNResultCallback 返回设置的结果

syncSetting

设置单位、语言、时间制式

参数

名称 类型 说明
length int 长度单位
0 公制，默认
1 英制
language int 语言类型
0 中文，默认
1 英文
hourFormat int 时间类型
0 24小时，默认
1 12小时
callback QNResultCallback 返回设置的结果

syncSitRemind

设置久坐提醒

参数

名称 类型 说明
sitRemind QNSitRemind 久坐提醒详情
callback QNResultCallback 返回设置的结果

syncHeartRateObserverMode

设置心率的监听模式

参数

名称 类型 说明
openFlag Booleann false 手动(默认)
true 自动
callback QNResultCallback 返回设置的结果

syncFindPhoneOpen

设置查找手机开关

只有开关打开时，手环屏幕才会出现该功能

参数

名称 类型 说明
openFlag Booleann false 手动(默认)
true 自动
callback QNResultCallback 返回设置的结果

syncCameraMode

设置相机模式

进入拍照模式后，触摸屏幕，手环会发出响应的信号。使用完成后必须退出拍照模式

参数

名称 类型 说明
openFlag Booleann false 退出拍照模式(默认)
true 进入拍照模式
callback QNResultCallback 返回设置的结果

syncHandRecognizeMode

设置抬腕识别模式

当打开抬腕识别后，抬腕时手环屏幕会自动亮屏

参数

名称 类型 说明
openFlag Booleann false 关闭(默认)
true 打开
callback QNResultCallback 返回设置的结果

setThirdRemind

ios专属

设置第三方提醒

设置第三方提醒时，倘若没有手环没有进行过配对，系统会弹框提醒用户进行配对，只有配对后才会有相应的提醒功能

参数

名称 类型 说明
thirdRemind QNThirdRemind 第三方提醒详情
callback QNResultCallback 返回设置的结果

reset

清除设置

主要用于快捷清除手环的一些设置

参数

名称 类型 说明
cleanSet QNCleanInfo 清除设置
callback QNResultCallback 返回设置的结果

reboot 恢复出厂设置

重启手环

参数

名称 类型 说明
callback QNResultCallback 返回结果

syncShortcutSetting

快捷设置

该方法仅支持ID为0003且版本号12后续版本(包含12版本)

参数

名称 类型 说明
config QNBandConfig 手环信息设置
callback QNResultCallback 返回结果

enterDfuMode

进入dfu模式

详情见OTA模式

参数

名称 类型 说明
callback QNResultCallback 返回结果

syncTodayHealthData

同步今日健康数据

今日数据与历史数据不可同时同步

参数

名称 类型 说明 泛型类型
callblock QNObjCallback 返回检测的结果 QNHealthData

syncHistoryHealthData

同步历史健康数据

今日数据与历史数据不可同时同步

参数

名称 类型 说明 泛型类型
callblock QNObjCallback 返回检测的结果 [QNHealthData]
