QNBleDeviceDiscoveryListener

QNBleDeviceDiscoveryListener

扫描设备监听回调类

方法

onDeviceDiscover

参数

名称 类型 说明
device QNBleDevice 扫描到的蓝牙设备

onStartScan

成功启动扫描后触发该方法

onStopScan

停止扫描后的回调

自动停止(定时)或调用stopScan停止后，都会触发该回调

onScanFail

没有扫描成功则回调(只有Android才会有这个回调)

参数

名称 类型 说明
code int 参考扫描状态码
