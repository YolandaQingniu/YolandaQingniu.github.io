QNWspScaleDataListener

QNWspScaleDataListener

wsp 设备数据监听器，该设备相关测量数据在这里回调，该 listener 继承于 QNScaleDataListener，此类已废弃

方法

wspRegisterUserComplete

wsp 设备注册用户成功，注册用户成功后设备会返回该用户在秤端的序列号

参数

名称 类型 说明
device QNBleDevice 蓝牙设备对象
user QNUser 用户对象

wspLocationSyncStatus

wsp 设备同步地理位置状态回调

参数

名称 类型 说明
device QNBleDevice 蓝牙设备对象
suceess Boolean 是否同步成功

wspReadSnComplete

wsp 设备获取SN码成功

参数

名称 类型 说明
device QNBleDevice 蓝牙设备对象
sn string SN码

wspGetLastDataHmac

wsp获取最近一次测量数据hmac,用于访客模式测量

参数

名称 类型 说明
device QNBleDevice 蓝牙设备对象
user QNUser 用户对象
