属性

名称 类型 说明
openFlag Boolean 开关
startHour int 提醒功能生效时间小时，设置范围为 0 - 23
startMinture int 提醒功能生效时间分钟，设置范围为 0 - 59
endHour int 提醒功能失效时间小时，设置范围为 0 - 23
endMinture int 提醒功能失效时间分钟，设置范围为 0 - 59
timeInterval int 时间间隔，隔多久提醒一次
week QNWeek 设置提醒的周期时间
