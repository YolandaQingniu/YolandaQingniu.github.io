  3. Add the development profile provided by Light Cow
  4. Read the documentation and demo carefully and access the SDK.
  5. Submit the developed APP to our company for testing and verification.
  6. Submit the application market online

Development Process

Before using the SDK, you need to call the initialization first: QNBleApi.initSdk

Code example:

//Android sample code
String configFilePath = "file:///android_asset/test123456789.qn";
QNBleApi.getInstance(context).initSdk("test123456789", configFilePath, new QNResultCallback() {
    @Override
    Public void onResult(int code, String msg) {
        Log.d("BaseApplication", "Initialization File" + msg);
    }
});

NSString *file = [[NSBundle mainBundle] pathForResource:@"123456789" ofType:@"qn"];
[[QNBleApi sharedBleApi] initSdk:@"123456789" firstDataFile:file callback:^(NSError *error) {
}];

The initialization method is the method that any device needs to call. The following are the workflows of different device types.

If you don't know what kind of equipment you have on your hands, you can refer to How to judge the device type in the FAQ
