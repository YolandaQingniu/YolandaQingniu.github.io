QNScaleStoreData

QNScaleStoreData

Light cattle store data objects, no APP is connected when measuring, and then connect to the APP after receiving it, it will receive such data.

Attributes

Name Type Description
weight double The weight corresponding to this stored data
measureTime Date Measurement time
mac String mac address

method

buildStoreData

Used to build stored data

Only support the acquisition of relevant storage data related information from Light Cow Cloud to build

Parameters

Name Type Description
weight double kg body weight
measureTime Date Measurement time
mac String Weighing end unique identifier
hmac String Related Encryption Information
callback QNResultCallback Building success

return value

Type: QNScaleStoreData

setUser

Set up measurement users

Name Type Description
user QNUser User data, used to produce measurement data

generateScaleData

Generate measurement data based on stored data and user data. If user data is not set, the method returns NULL.

return value

Type: QNScaleData

Returns NULL when an error occurs
