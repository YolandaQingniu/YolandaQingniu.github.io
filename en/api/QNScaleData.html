QNScaleData

QNScaleData

Light cow body fat is called measurement data, including body weight, BMI, body fat percentage and other data.

Attributes

Name Type Description
user QNUser Measured users, including userId, gender, birthday, height, etc.
measureTime Date Measured time, accurate to seconds
hmac String Characteristic identification of data

method

getItem

Get a single metric, or NULL if there is no specified type

Parameters

Name Type Description
type int Index type, reference schedule

return value

Type: QNScaleItemData

A single indicator object, including the value of this indicator, the English name, whether it meets the standard, etc.

getItemValue

Get the value of a single metric and only return the value of this metric

Parameters

Name Type Description
type int Index type, reference schedule

return value

Type: double

Returning the type of floating point number, the value of some indicators is actually int type, the APP needs to convert itself.

getAllItem

Return all the indicators in the form of a list. If the measurement data is invalid, the value of the other indicator data is 0 except for the weight and BMI indicators.

return value

Type: List<QNScaleItemData>

List of indicators, the order will be sorted by TYPE, if the APP needs to redefine the order, you need to sort by yourself.

setFatThreshold

Body fat change control

Used to control changes in body fat and alleviate the problem of excessive body fat

Precautions for use:
1. This method can only be used if there is no major change in body data and weight;
2. The accessor must save the characteristic identifier hmac of the data for the body Lipid change control;
3. It is necessary to obtain the relevant index value after calling this method for control, otherwise data confusion will occur.

Parameters

Name Type Description
hmac String The data characteristic identifier of the last valid measurement data (that is, the measured body fat measurement data) cannot be nil
threshold Double The maximum error control of body fat, this value is only the approximate range of control, not precise control.
For example, if the value is 2, then when the measured body fat differs from the previous time by more than 2, the measured body fat will be controlled by about 2 degrees from the last measured body fat
callback QNResultCallback Setting success
