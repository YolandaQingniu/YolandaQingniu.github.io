QNUser

QNUser

Light cattle user object

Attributes

Name Type Description
UserId String User unique identifier, cannot be empty, unique for each user, related to stored data
height int height, unit cm, minimum 40, maximum 240
Gender String Gender, Male: male, Female: female
Birthday Date Birthday, used to calculate age, date accuracy is accurate to the day, the calculated age range is 3~80, beyond this range, equal to the upper or lower limit.
AthleteType int Whether it is an athlete mode (0 is a normal algorithm, 1 is an athlete algorithm), this field only takes effect when the age is greater than or equal to 18 years old
clothesWeight double Unit: KG
Clothing weight. When this value is set, the measured body weight value will return the value minus the weight of the clothing.
The weight of the clothes should not exceed half of the weight, otherwise the excess value will be ignored
