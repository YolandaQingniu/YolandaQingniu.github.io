Scale event definition

Scale event definition

event value meaning Bluetooth properties
EVENT_WIFI_BLE_START_NETWORK 1 WiFi Bluetooth dual-mode device starts to distribute network connected
EVENT_WIFI_BLE_NETWORK_SUCCESS 2 WiFi Bluetooth dual-mode device networked successfully connected
EVENT_WIFI_BLE_NETWORK_FAIL 3 WiFi Bluetooth dual-mode device networking failed connected
EVENT_REGIST_USER_SUCCESS 4 WSP scale exclusive, registered user success connected
EVENT_REGIST_USER_FAIL 5 WSP scale exclusive, registered user failed connected
EVENT_VISIT_USER_SUCCESS 6 WSP scale exclusive, successful access to users connected
EVENT_VISIT_USER_FAIL 7 WSP scale exclusive, failed to access user connected
EVENT_DELETE_USER_SUCCESS 8 Exclusive for WSP scales, successful user deletion connected
EVENT_DELETE_USER_FAIL 9 WSP scale exclusive, failed to access user connected
